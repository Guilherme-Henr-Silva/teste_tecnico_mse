# Cadastro de Fornecedores

## Funcionalidades do projeto
- Cadastro de fornecedores
- Validação de CNPJ
- Listagem
- Edição
- Exclusão (CRUD completo)

## Como executar

1. Instalar XAMPP
2. Iniciar Apache e MySQL
3. Criar banco via phpMyAdmin
4. Colocar projeto em htdocs
5. Acessar:

http://localhost/teste_tecnico_mse

## Banco de dados

O dump do banco está disponível na pasta database.
Para importar:

1. Acessar phpMyAdmin
2. Criar banco fornecedores_db
3. Ir em Importar
4. Selecionar o arquivo fornecedores_db.sql
